- `n - 1`：从最右边的 1 对应的位置开始，对后面的部分取反

    （1100 - 1 = 1011）

- `n & n - 1`：将最右边的 1 变为 0

    （1100 & 1011 = 1000）

```cpp
class Solution {
public:
     int NumberOf1(int n) {
         int count = 0;
         while (n != 0)
         {
             ++count;
             n = n & (n - 1);
         }
         return count;
     }
};
```
